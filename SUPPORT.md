# Support

## Community Support

This project is provided **as-is** and is supported on a **best-effort, community basis** through GitHub Issues. There is no SLA or guaranteed response time.

---

## Filing a Bug Report

If you encounter a problem with the script, please [open a GitHub Issue](../../issues/new?template=bug_report.md).

To help diagnose your issue quickly, please include:

| Information | Example |
|---|---|
| **Windows version & build** | Windows 11 24H2, build 26100.3775 |
| **Intune DO policy configuration** | DODownloadMode = 2 (Group), MCC host configured |
| **Full script output string** | `DOServiceRunning,DOmode:NotKnownCacheEmpty,...` |
| **Any error messages** | Full text of any PowerShell error output |
